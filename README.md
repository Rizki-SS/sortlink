# Sortlink Service with multiple database support

This Sortlink Service offers robust support for multiple database systems, providing unparalleled flexibility in data storage and management. It leverages a consistent hashing method for efficient and reliable generation of short URLs, ensuring unique and collision-resistant links while allowing seamless integration with diverse database backends.

## Features

- [x] Link management feature
- [x] JWT authentication for secure access
- [x] Multiple database support
- [x] Consistent hashing method for efficient and reliable generation of short URLs
- [x] Scalable architecture for high traffic
- [x] Redis cache to improve query performance
- [ ] SEO features (e.g., UTM parameters)
- [ ] Geolocation-based routing
- [ ] Traffic balancing

## Getting Started
To get started with this template, simply paste this command into your terminal:
```bash
bun create elysia ./elysia-example
```

## Development
To start the development server run:
```bash
bun run dev
```

Open http://localhost:3000/ with your browser to see the result.
